          Key-values passed to GAM

          The engine stitches these into the GAM tag's cust_params. Set up matching key-value targeting + line items in Google Ad Manager.

          KeyMeaning
          hb_pbWinning price bucket (after floor bias) — drives price-priority line items
          hb_bidderWinning SSP code
          hb_pb_<bidder>Per-bidder price (send-all-bids)
          hb_bidder_<bidder>Per-bidder code (send-all-bids)

          Fallback: when Prebid returns no demand, the engine calls GAM with hb_pb=0.00 & hb_bidder=none so your house line item serves. Line items/creatives are configured in GAM itself (GAM API automation is a later phase).
